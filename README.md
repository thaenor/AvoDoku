# AvoDoku

AvoDoku is a Sudoku game specifically designed for accessibility, targeting elderly users or individuals who require high-contrast interfaces and ease of use on laptop computers.

The project prioritizes readability, large interactive targets, and simple navigation while providing a full-featured Sudoku experience.

## Features

-   **Accessibility First**: High-contrast black-on-white design with clear error and selection states.
-   **Notes Mode**: "Pencil mark" annotations with a realistic handwriting font (Caveat).
-   **Smart Difficulty**: Three difficulty levels (Easy, Medium, Hard) generated by a randomized backtracking algorithm.
-   **Helper Tools**: Intelligent Hint system, Undo/Redo history, and Auto-check for victory.
-   **Localization**: Fully translated into European Portuguese (pt-PT).
-   **Persistence**: Automatically saves progress and game history so you never lose your game.

## Tech Stack

-   **Frontend**: React + TypeScript
-   **Build Tool**: Vite
-   **Styling**: Tailwind CSS
-   **State Management**: Zustand
-   **Fonts**: Inter (System UI) & Caveat (Notes)

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the development server:
    ```bash
    npm run dev
    ```
